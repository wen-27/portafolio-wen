Liga BetPlay

Este proyecto en Python te permite consultar y visualizar datos de la liga colombiana, como posiciones, fechas de partido y estadísticas, ¡todo desde la consola!

Características

  • Acceso a información actualizada de la Liga BetPlay (Dimayor).
  • Consulta de resultados, tabla de posiciones y fechas programadas.
  • Fácil de ejecutar y personalizar.

Tecnologías usadas

  • Python
  • Requests (o similar) para solicitudes web
  • Git y GitHub

Logros

  • Automatización de la consulta de resultados de la liga.
  • Implementación de solicitudes web para obtener datos actualizados.
  • Optimización del código para hacerlo modular y reutilizable.
  • Publicación del proyecto en GitHub como portafolio profesional.
