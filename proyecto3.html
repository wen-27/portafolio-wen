Proyecto Mapa HTML

Este proyecto muestra un mapa interactivo usando solo HTML y CSS, ideal para visualización de ubicaciones sin necesidad de librerías externas.

Características

  • Mapa responsivo y sencillo
  • Sin dependencias externas
  • Fácil de usar y personalizar

Tecnologías usadas

  • HTML5
  • CSS puro

Logros

  • Diseñé un mapa totalmente responsivo sin necesidad de frameworks.
  • Conseguí una interfaz ligera y rápida, optimizada solo con HTML y CSS.
  • Mejoré mis habilidades en maquetación y posicionamiento con CSS.
  • Publiqué el proyecto en GitHub como parte de mi portafolio profesional.
